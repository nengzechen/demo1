## 🚀 如何使用

### 快速启动

```bash
# 1. 进入项目目录
cd play-demo

# 2. 运行项目
sbt run

# 3. 等待启动（约30秒）
# 看到：(Server started, use Enter to stop...)

# 4. 访问API
http://localhost:9000
```

### 运行测试

```bash
# 方式1：使用Postman
1. 启动项目（sbt run）
2. 打开Postman
3. 导入：postman/UserPermissionAPI.postman_collection.json
4. 设置：baseUrl = http://localhost:9000
5. 执行：Run Collection

# 方式2：使用cURL
curl -X POST http://localhost:9000/api/management/users \
  -H "Content-Type: application/json" \
  -d '{"username":"test","password":"pass123","email":"test@test.com"}'
```
